## 服务器配置参考

<b style="color:red">CPU最大支持64核，否则磁盘IO在CPU满载情况下无法满足高并发</b>

| CPU    | 内存   | 硬盘      | 存储量     |
| ------ | ------ | --------- | ---------- |
| 4      | 16     | 500GB SSD | 200亿      |
| 8      | 16     | 500GB SSD | 200亿      |
| 16     | 32     | 1TB SSD   | 400亿      |
| 32     | 32     | 2TB SSD   | 800亿      |
| 64     | 32     | 4TB SSD   | 1600亿     |
| 64     | 64     | 8TB SSD   | 3000亿     |
| 分布式 | 分布式 | 分布式    | 理论上无限 |
